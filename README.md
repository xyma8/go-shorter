
# go-shorter
---
Сокращатель ссылок нового поколения

---
Переменные окружения (env):

| Название             | Описание                                                                                                                                                                                         | Пример                   | По умолчанию                 |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------ | ---------------------------- |
| `DB_TYPE`            | Тип БД (PostgreSQL или SQLite)                                                                                                                                                                   | postgresql, sqlite       | sqlite                       |
| `APP_PORT`           | Порт, на котором запустится сервис                                                                                                                                                               | 8080                     | 8080                         |
| `UI_FILES_DIR`       | Путь до статических файлов UI                                                                                                                                                                    | ./public                 | ./public (как в репозитории) |
| `BACKEND_PROTOCOL`   | Протокол URL, используется для формирования в ответе полного адреса с короткой ссылкой.                                                                                                          | http, https              | -                            |
| `BACKEND_HOST`       | Хост в URL, используется для формирования в ответе полного адреса с короткой ссылкой.<br>Шаблон и полный пример:<br>*{BACKEND_PROTOCOL}://{BACKEND_HOST}/?????*<br>*http://localhost:8080/abc2e* | localhost:8080           | -                            |
| `PERMUTE_SECRET_KEY` | Ключ, используемый для алгоритма перемешивания последовательности id и коротких ссылок. Задается один раз в начале и менять при наличии данных нельзя.                                           | your-super-secret-key123 | -                            |
| `POSTGRES_USER`      | Имя пользователя БД                                                                                                                                                                              | admin                    | -                            |
| `POSTGRES_PASSWORD`  | Пароль пользователя БД                                                                                                                                                                           | strongpassword           | -                            |
| `POSTGRES_HOST`      | Адрес БД postgres                                                                                                                                                                                | 0.0.0.0                  | 0.0.0.0                      |
| `POSTGRES_PORT`      | Порт БД                                                                                                                                                                                          | 5432                     | 5432                         |
| `POSTGRES_DB`        | Название БД                                                                                                                                                                                      | goshorter                | goshorter                    |
| `POSTGRES_SSLMODE`   | SSL mode                                                                                                                                                                                         | disable, enable          | disable                      |
