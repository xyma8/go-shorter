# go-shorter

---

Сокращатель ссылок нового поколения

---

Переменные окружения (env):

| Название            | Описание                                                                                                                                                                                         | Пример         | По умолчанию |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | -------------- | ------------ |
| `POSTGRES_USER`     | Имя пользователя БД                                                                                                                                                                              | admin          | -            |
| `POSTGRES_PASSWORD` | Пароль пользователя БД                                                                                                                                                                           | strongpassword | -            |
| `POSTGRES_HOST`     | Адрес БД postgres                                                                                                                                                                                | 0.0.0.0        | 0.0.0.0      |
| `POSTGRES_PORT`     | Порт БД                                                                                                                                                                                          | 5432           | 5432         |
| `POSTGRES_DB`       | Название БД                                                                                                                                                                                      | goshorter      | goshorter    |
| `POSTGRES_SSLMODE`  | SSL mode                                                                                                                                                                                         | disable        | disable      |
| `BACKEND_PROTOCOL`  | Протокол URL, используется для формирования в ответе полного адреса с короткой ссылкой.                                                                                                          | http           | -            |
| `BACKEND_HOST`      | Хост в URL, используется для формирования в ответе полного адреса с короткой ссылкой.<br>Шаблон и полный пример:<br>_{BACKEND_PROTOCOL}://{BACKEND_HOST}/?????_<br>_http://localhost:8080/abc2e_ | localhost:8080 | -            |
